media queries help in making responsive websites

each media feature falls to resole to either true or false

examples

  1. max-width
  2. min-width
  3. orientation (potrait)
  4. screen
  5. print
media queries can be combined using logical operators

@media (min width : 768px) and (max-width: 991px)

always start with base styles in css

careful not to overlap range boundaries
